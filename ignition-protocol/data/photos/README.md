# Progress Photos

Store or link weekly photos here. Consistency matters.

- Angles: front, side, back.
- Conditions: same lighting, distance, time of day, camera.
- Naming: `week-##-front.jpg`, `week-##-side.jpg`, `week-##-back.jpg` (or links if stored in cloud).
- Note photo weeks in weekly summaries and link them in `analysis/weekly_trends.md`.
